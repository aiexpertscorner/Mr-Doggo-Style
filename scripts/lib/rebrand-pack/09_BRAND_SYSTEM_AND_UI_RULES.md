# BRAND SYSTEM AND UI RULES

Use the supplied PupWiki logo direction and apply it calmly.

## Desired style
- modern
- trustworthy
- clean
- editorial
- dog-friendly without being childish
- readable on mobile and desktop

## System work
Update or define:
- global color tokens
- type scale
- section spacing rhythm
- buttons
- cards
- badges
- callouts
- comparison tables
- author/info boxes
- disclosure blocks

## Strong rule
Do not overdecorate.
Avoid making the site feel AI-generated, cartoonish, or affiliate-thin.

## Logo usage
Use:
- PupWiki
- PupWiki.com where needed

Avoid:
- Pup Wiki
- inconsistent capitalization
- overuse of paw graphics outside the main lockup
