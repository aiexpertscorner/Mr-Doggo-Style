# SEO, SCHEMA, AND SITEMAP RULES

Refine the SEO layer in Astro-safe ways.

## Check for
- title helpers
- layout-level metadata
- Open Graph helpers
- canonical builders
- JSON-LD/schema components
- sitemap integration in Astro config or route generation
- robots behavior
- manifest/site URL definitions

## Update
- brand name
- domain
- organization name
- publisher strings
- logo references
- social preview references
- title patterns
- meta description tone
- page-type metadata quality

## Page-type-specific logic
Ensure metadata logic is strong for:
- homepage
- breed pages
- blog pages
- names pages
- cost calculator pages
- category pages
- trust pages

## Rule
Keep ranking risk low.
Do not change slugs unless migration handling is clear and worth it.
