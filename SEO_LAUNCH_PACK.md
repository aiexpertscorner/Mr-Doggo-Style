Build these links into every new post:

- `/blog/best-dog-food-large-breeds` — link from ALL food posts
- `/blog/best-indestructible-dog-toys` — link from ALL toy posts
- `/blog/best-dog-beds-orthopedic` — link from ALL bed posts
- `/blog/best-no-pull-harnesses` — link from ALL training posts
- `/blog/best-joint-supplements-dogs` — link from food posts + bed posts

### Supporting Pages (link out to hubs)
- Breed pages → link to relevant category hubs (3-4 links per breed page)
- Comparison posts → link to individual product review posts
- Food posts → cross-link to each other + supplement posts

### Anchor Text Rules
- Use descriptive anchor text matching the target page's keyword
- ✅ "best orthopedic dog bed for large breeds"
- ❌ "click here" or "read more"
- ❌ Same anchor text on every link

### Recommended: Add "Related Reviews" blocks to each post
```astro
<RelatedPosts currentTags={tags} currentSlug={Astro.url.pathname} />
```
This auto-generates 3 related posts based on matching tags — set tags carefully.

## 3. SCHEMA MARKUP CHECKLIST

### Every Blog Post Must Have:
- [x] Article schema (headline, datePublished, dateModified, author, image)
- [x] BreadcrumbList schema
- [x] FAQPage schema on posts with FAQ sections
- [ ] Review schema with `reviewRating` on each product reviewed
- [ ] ItemList schema on roundup posts (list the top picks)

### To Add Review Schema to Each Product:
Use the `ProductReviewCard` component from the upgrade pack — it auto-generates Review schema.

For existing posts using the old `ProductCard`, add this after each product heading:

```json
{
  "@context": "https://schema.org",
  "@type": "Review",
  "itemReviewed": {
    "@type": "Product",
    "name": "KONG Extreme Dog Toy",
    "offers": {
      "@type": "Offer",
      "price": "17.99",
      "priceCurrency": "USD",
      "url": "https://www.amazon.com/dp/B0002AR0II/?tag=aiexpertscorn-20"
    }
  },
  "reviewRating": { "@type": "Rating", "ratingValue": "9", "bestRating": "10" },
  "author": { "@type": "Organization", "name": "Mr. Doggo Style" },
  "reviewBody": "The gold standard for tough dog toys..."
}
```

### Breadcrumb Schema (auto-generated by BlogLayout upgrade):
- Home › Blog › Category › Post Title
- All levels clickable and linked

### Test Your Schema:
1. Google Rich Results Test: https://search.google.com/test/rich-results
2. Schema.org Validator: https://validator.schema.org
3. Aim for "no errors" on all pages

## 4. META TITLE OPTIMIZATION

### Formula for Product Review Posts:
`Best [Product] for [Audience] [Year]: [Hook]`

Examples:
- ✅ Best Dog Food for Large Breeds 2026: 7 Vet-Approved Picks
- ✅ Best Indestructible Dog Toys 2026: 5 That Actually Survived Our Tests
- ❌ The Best Dog Food Reviews (vague, no year, no hook)

### Formula for Comparison Posts:
`[Product A] vs [Product B]: [What Readers Want to Know]`

Examples:
- ✅ KONG Extreme vs Goughnuts MAXX: Which Lasts Longer?
- ✅ Chewy vs Amazon for Dog Food: Which Is Actually Cheaper?

### Title Length Rules:
- Aim for 50-60 characters (fits in Google SERP without truncation)
- Check length at: https://www.highervisibility.com/seo/tools/serp-snippet-optimizer/

## 5. META DESCRIPTION OPTIMIZATION

### Formula:

- ✅ "We evaluated 30+ large breed formulas for protein quality, joint support, and value. Here are the 7 that actually stand out. Find your dog's perfect food."

## 6. IMAGE SEO CHECKLIST

For every image added:
- [ ] Descriptive filename: `kong-extreme-black-rubber-dog-toy.jpg` not `IMG_1234.jpg`
- [ ] Alt text: describes the image content for accessibility + SEO
  - ✅ `alt="KONG Extreme black rubber dog chew toy on white background"`
  - ❌ `alt="dog toy"` or `alt="image"`
- [ ] Compress before upload: use Squoosh (https://squoosh.app) — target <150KB per image
- [ ] WebP format when possible (better compression than JPEG/PNG)
- [ ] Lazy loading: already implemented with `loading="lazy"` in ProductCard

## 7. PAGE SPEED CHECKLIST

Astro SSG + Tailwind already gives you an excellent baseline. Additional improvements:

### Implement in astro.config.mjs:
```js
import { defineConfig } from 'astro/config';
import tailwind from '@astrojs/tailwind';
import sitemap from '@astrojs/sitemap';
import compress from 'astro-compress'; // npm install astro-compress

export default defineConfig({
  site: 'https://mrdoggostyle.com',
  integrations: [
    tailwind(),
    sitemap(),
    compress(), // Compresses HTML, CSS, JS, images at build time
  ],
  build: {
    inlineStylesheets: 'auto', // Inline small CSS, reducing requests
  },
});
```

### Install: `npm install astro-compress`

### Core Web Vitals Targets:
- LCP (Largest Contentful Paint): < 2.5s ✅ (SSG achieves this)
- FID/INP (Interaction to Next Paint): < 200ms ✅ (minimal JS)
- CLS (Cumulative Layout Shift): < 0.1 ✅ (use width/height on images)

### Test at: https://pagespeed.web.dev
